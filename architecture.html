Component Deep Dive

Technical specifications for core system components.

The Agent Layer

Each agent is a specialized LLM chain with a specific prompt, toolset, and knowledge base.

  • Medication Agent: Connected to FDA drug databases. Strict "no-prescription" hard constraint.
  • Medical Info Agent: RAG-based retrieval from curated medical journals (PubMed, etc.).
  • Empathy Agent: Fine-tuned on therapeutic conversation datasets. Prioritizes sentiment analysis.

The Safety Guardian

A deterministic middleware that intercepts all agent inputs and outputs.

  • Input Guard: PII redaction, intent classification (emergency detection).
  • Output Guard: Hallucination check (citation verification), toxicity filter, medical advice boundary check.
  • Escalation Protocol: Automatic routing to human support if confidence score drops below 0.95.

Integration Standards

Built for interoperability with existing hospital systems.

  • FHIR R4: Native support for Fast Healthcare Interoperability Resources.
  • SMART on FHIR: Secure authorization and launch context.
  • HL7 v2: Legacy support via adapter layers.
