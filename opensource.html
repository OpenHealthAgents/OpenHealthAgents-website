Getting Started

It's easy to make your first contribution.

  1. Fork the Repository

    Visit our GitHub and fork the core repository to your own account.

  2. Pick an Issue

    Look for issues tagged good-first-issue or help-wanted. We have tasks for all skill levels.

  3. Submit a Pull Request

    Push your changes and submit a PR. Our maintainers will review your code and guide you through the merge process.
